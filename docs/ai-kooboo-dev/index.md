# AI Kooboo 开发

这一栏用于专门存放 AI 参与 Kooboo 开发时最值得直接复用的内容。

## 更适合解决

- 给 AI 一段能直接开工的上下文
- 找到稳定好用的提示词
- 找到已经验证过的 skills

## 常见对象

- 项目接管提示词
- 功能开发提示词
- review 提示词
- skills

## 不要混淆

- 这一栏解决的是 AI 的工作入口，不是对象定义页
- 运行时 API 细节仍然应回到 `KScript` 和内容系统页面

## 下一步阅读

- [开箱提示词](/ai-kooboo-dev/prompts)
- [推荐 Skills](/ai-kooboo-dev/skills)

重点只有两类：

- 可直接开箱使用的提示词
- 经过验证、值得优先使用的 skills

## 为什么要单独做这一栏

很多 Kooboo 项目并不是“AI 自己猜一套写法”就能顺利完成的。更有效的方式是给 AI 一套稳定入口，让它先获得正确上下文，再开始写代码。

这个栏目就是为了解决这个问题：

- 减少每次重新解释背景
- 让 AI 优先读取正确的项目和规则
- 让已验证过的 skills 成为默认优先项

## 阅读顺序

1. [开箱提示词](/ai-kooboo-dev/prompts)
2. [推荐 Skills](/ai-kooboo-dev/skills)

## 适用场景

- 你准备让 AI 接手一个 Kooboo 项目
- 你希望 AI 在修改前先读对项目上下文
- 你想把好用的 skills 沉淀下来，方便重复使用
